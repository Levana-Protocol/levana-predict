### Usage

#### Enabling Corepack:

```
corepack enable pnpm
```

> This will automatically install the pnpm version used by the project.

#### Installing dependencies:

```
pnpm install
```

#### Running the app (on port 3000):

```
pnpm dev
```

#### Adding dependencies

```
pnpm add <pkgs>
```

> For dev dependencies, use the `-D` flag.
